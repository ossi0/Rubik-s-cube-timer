# Rubik-s-cube-timer
Timer for speedcubing with personal statistics.

-------------------------------------------------

Starting guide:

1. Edit cubeapp.BAT file and change path to corresponding path and save
2. Copy cubeapp.BAT to desktop
3. Install Node.js -- https://nodejs.org/en/
4. On command line go to this app's directory and write:
-- npm install express
-- npm install
-- npm install body-parser
